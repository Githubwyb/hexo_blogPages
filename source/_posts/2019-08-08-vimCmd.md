# 常用命令

## 行号

```shell
    :set number     # 显示行号
    :set nonumber   # 取消显示行号
```

## 安装

### 文件树插件 NERDTree

克隆仓库并查看版本

```shell
    git clone https://github.com/scrooloose/nerdtree.git ~/.vim/bundle/nerdtree
    cd ~/.vim/bundle/nerdtree   # 到目录查看版本号
    git tag                     # 查看版本号
    git checkout xxxx           #切换到相应版本
```
