# 写文档

## 空格

```markdown
    &nbsp;
```

## 超链接

代码

```markdown
    [我的博客][1]

    [1]: https://githubwyb.github.io/
```

效果

[我的博客][1]

[1]: https://githubwyb.github.io/

## 文内跳转

代码

```markdown
    <span id = "test">跳转到这里</span>

    [跳转](#test)
```

效果

<span id = "test">跳转到这里</span>

[跳转](#test)
