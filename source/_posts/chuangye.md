# 河大 vs 河师大

------------------

|                    | 河大             | 河师大                   |
| ------------------ | ---------------- | ------------------------ |
| 名气               | **较大**         | 较小                     |
| 师范名气           | 较小             | **较大**                 |
| 关系               | 很需要(老王说)   | **稍微需要(学校老师说)** |
| 考试难度           | 较大             | **较小**                 |
| 学习难度           | 都要学           | 都要学                   |
| 资料获取           | 较为麻烦         | **较为简单**             |
| 联系导师           | 差不多           | 差不多                   |
| 联系学长学姐       | 要找辅导员       | **到处都是**             |
| 心里因素           | **离家比较近**   | 离家比较远               |
| 宝贝不愿考虑的因素 | **我能直接找人** | 我能间接找人             |
