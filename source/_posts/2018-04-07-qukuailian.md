# 区块链

- 定义: 基于共识协议和加密算法的去中心化分布式账本系统
- 防篡改 透明 去中心
- 分布式 **可信** 全共识
- 脱媒: 去掉中介
    - 节点到节点
- 发言加编号来统计人数--区块链

| 微信组局                                                  | 在区块链里的概念                         |
| --------------------------------------------------------- | ---------------------------------------- |
| 接龙发帖                                                   | 链式数据结构                             |
| 规则：每个人发帖 = 上一个帖子内容 + 下一个编号 + 自己名字 | 共识机制，根据严格的规则和公开的协议形成 |
| 规则定下来后，大家自发登记                                | 去中心化，没有任何单一用户能够掌控它     |
| 在微信里记录登记情况                                      | 点对点对等网络                           |
|                                                           |
